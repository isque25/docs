## Setting up

Create your api key in the [dashboard](https://dashboard.agenttoolkit.ai). Then, head to quickstart to get started.

## Overview

Agent Toolkit allows you to quickly add tools which allow your agents to connect to web content easily.

## Search

Search allows agents to perform web searched. The search results are returned, along with an optional summary of the generated results


## Extract

Extract allows you to get content from a specific url. You may also optionally return a list of links or images found in the page.
